Viktor

Viktor: A Task Management Assistant

Viktor is a task-manager chatbot built in Java, inspired by the hit Netflix show Arcane's Viktor. It features persistent storage, and boasts a structured object-oriented design, making it both scalable and maintainable. Viktor leverages Gradle for dependency management and testing, ensuring smooth builds and integration. A JavaFX graphical user interface further enhances user interaction.

Viktor isn't just a task manager; it's an evolving project that reflects both efficiency and innovation!
