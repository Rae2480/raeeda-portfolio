Viktor: A Smart and Efficient Task Management Assistant

Viktor is a chatbot built in Java, inspired by the hit Netflix show Arcane's Viktor. Designed as an intelligent task manager, it allows users to efficiently organize their tasks, categorised as to-dos, deadlines, and events with a simple yet powerful command-based interface. The bot features persistent storage, and boasts a structured object-oriented design, maintaining clean separation of concerns, making it both scalable and maintainable.

Developed with Java, Viktor leverages Gradle for dependency management and testing, ensuring smooth builds and integration. A JavaFX graphical user interface further enhances user interaction, transforming Viktor from a command-line tool into an intuitive and visually engaging assistant. Beyond basic task tracking, future improvements include natural language processing for more flexible commands and potential integration with external APIs for productivity enhancements.

Viktor isn't just a task manager—it’s an evolving project that reflects both efficiency and innovation.
